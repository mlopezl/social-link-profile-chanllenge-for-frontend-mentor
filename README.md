## Overview
This project is a **social links profile card** that displays a user's information, avatar, location, and links to multiple social media platforms.

The interface features:
- A clean and modern profile UI  
- A centered layout with social buttons  
- Interactive hover states on social links  
- Fully responsive design for both mobile and desktop  

## The Challenge
Users should be able to:

- View the layout clearly on **both desktop and mobile screens**  
- See **hover state** for the social media buttons  
- Navigate through social media links in a structured list

## My Process
1. Created the project structure using **semantic HTML sections**.  
2. Imported a **custom font with `@font-face`** for better typography control.  
3. Defined the color theme using **CSS variables**, allowing an organized and scalable palette.  
4. Built the layout using **Flexbox** to center and align all elements.  
5. Styled interactive elements with **hover effects** for better user experience.  
6. Made the design responsive using **media queries** to adapt to smaller screens.
