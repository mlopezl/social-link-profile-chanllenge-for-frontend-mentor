## Resumen
Este proyecto es una **tarjeta de perfil con enlaces sociales** que muestra la información de un usuario, avatar, ubicación y enlaces a múltiples plataformas sociales.

La interfaz incluye:
- Una UI moderna y limpia  
- Un diseño centrado con botones para redes sociales  
- Efectos hover interactivos en los enlaces  
- Diseño totalmente responsive para computadoras y dispositivos móviles  

## El Desafío
Los usuarios deberían poder:

- Visualizar el diseño correctamente en **pantallas de escritorio y móviles**  
- Ver el **estado hover** en los botones de redes sociales  
- Navegar a través de los enlaces sociales de forma clara y estructurada

## Mi Proceso
1. Creé la estructura del proyecto utilizando **secciones semánticas en HTML**.  
2. Importé una **fuente personalizada mediante `@font-face`** para un mejor control tipográfico.  
3. Definí el tema de color usando **variables CSS**, permitiendo una paleta organizada y escalable.  
4. Construí el layout utilizando **Flexbox** para centrar y alinear todos los elementos.  
5. Apliqué **efectos hover** a los elementos interactivos para mejorar la experiencia de usuario.  
6. Hice el diseño responsive utilizando **media queries** para adaptarlo a pantallas más pequeñas.
